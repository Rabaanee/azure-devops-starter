# Azure DevOps Starter

A simple starter project to practice DevOps fundamentals:
- Version control with Git/GitHub
- CI pipeline that runs tests automatically
- Clean repo structure and documentation

## What it does
Runs a small Python script and a basic test suite.

## Tech
- Python
- Pytest
- GitHub Actions (CI)
